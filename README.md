# A simple python webservice for DETA

Starting point for lots of cool stuff...

## Requirements

 * **Flask** https://flask.palletsprojects.com/

## Local run configuration

Parameters: `-m flask run`

## Configure

```shell script
deta login
deta new --python <MICRO_NAME>
```

## Deploy

### Manual deploy

```shell script
deta deploy
```

### Automatic deploy

Via github action.
